Configuración de facturas

Próxima fecha de facturación

La fecha de facturación se usa para determinar cuándo se van a generar las facturas en la plataforma. La fecha se actualiza automáticamente una vez el sistema detecta que se ha llegado a la fecha, de manera que se tiene que tener cuidado cuando se cambia la fecha, porque puede activar la generación de facturas un día que no se debería.
